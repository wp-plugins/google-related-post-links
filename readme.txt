Displays a list of related posts and searches by Google

== Description ==

This plugin is uses [Google Related Links Demo page](http://relatedlinks.googlelabs.com/config/demo) to process related links for your posts. The links are highly relevant as it uses googles unmatched search technology to get the related links. The links are highly SEO friendly and can be easily indexed by search engine robots.  

[Click here for Detailed Instructions and Screenshots](http://letusbuzz.com/google-related-post-links/)

Features of the plugins

*   Highly relevant related post results (upto 8 links).
*   All displays a list of related searches (Can be enabled/disabled from plugin options)


If you think that some features are missing then suggest me I will be happy to add those features to the plugin. Your suggestion is always appreciated.
[Comment on this page ](http://letusbuzz.com/google-related-post-links/) to suggest a feature.

== Installation ==

1. Upload `google-relates-post-links` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Goto the Settings > Google Related Posts redirect to change setting
